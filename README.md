# Dra. Claudia Niño - Sitio Web Profesional

Sitio web profesional para la Dra. Claudia Niño, odontóloga especializada en estética dental en Bogotá, Colombia.

## Características

- **Diseño Moderno y Minimalista**: Interfaz limpia y elegante que refleja profesionalismo
- **Totalmente Responsive**: Optimizado para dispositivos móviles, tablets y escritorio
- **Secciones Principales**:
  - Header atractivo con mensaje sobre transformación de sonrisas con tecnología avanzada
  - Servicios especializados:
    - Ortodoncia Invisible
    - Implantes Dentales
    - Diseño de Sonrisa
    - Estética Dental
  - Beneficios y razones para elegir la práctica
  - Llamado a la acción claro para agendar citas

## Tecnologías Utilizadas

- HTML5
- CSS3 (con variables CSS y Flexbox/Grid)
- Google Fonts (Montserrat y Playfair Display)
- Diseño responsive con media queries

## Cómo Usar

1. Simplemente abra el archivo `index.html` en su navegador web
2. No se requiere instalación ni configuración adicional
3. El sitio es estático y puede ser alojado en cualquier servidor web

## Personalización

Para personalizar el sitio:
- Edite `index.html` para modificar el contenido
- Modifique `styles.css` para ajustar colores, fuentes y estilos
- Las variables CSS están definidas en `:root` para facilitar cambios de tema
